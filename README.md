## Dependencies
- Ruby 2.7
- Rails 6.0
- Json Web Token (JWT)
- PostgreSQL
- Faker
- Serializers

## Installation
- Clone this project

        git clone https://github.com/vedtra/sehatq-backend-test.git
        
- Install dependencies

        bundle install

- Create database

        rails db:create
        
- Run migration

        rails db:migrate

- Db seed

        rails db:seed

- Run this project
        
        rails s
